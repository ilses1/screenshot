# 多屏遮罩与跨屏截图使用说明

## 快捷键与流程
- 开始遮罩：按 `F1`（进入多屏遮罩模式，鼠标事件默认可穿透到底层应用）
- 进入框选：再次按 `F1`（进入框选输入模式）
- 框选：按住左键拖拽可跨屏选择区域，松开后显示 `✓/✕` 按钮
- 确认：点击 `✓` 或按 `Enter` 完成截图（写入剪贴板，并按设置项自动保存/打开编辑器）
- 取消：按 `Esc` 或右键取消（立即销毁所有遮罩/输入窗口，不产出截图）

## 配置项
### 遮罩透明度（maskAlpha）
- 范围：`0.60 ~ 0.80`
- 默认：`0.70`
- 入口：设置页的“遮罩透明度”滑块

## 交互与表现
- 多屏遮罩：每个物理显示器各自创建全屏遮罩窗口（置顶、不抢焦点）
- 跨屏框选：由覆盖虚拟桌面的输入窗口捕获拖拽，实时广播选区给各遮罩窗口进行绘制
- 事件穿透：
  - 遮罩模式：遮罩窗口忽略鼠标事件并 forward 到底层
  - 框选模式：输入窗口接管鼠标用于框选与确认/取消

## 已知限制
- macOS：需要授予“屏幕录制”权限，否则无法抓取屏幕内容
- Linux Wayland：部分桌面环境对截屏能力限制较多，可能无法抓取或得到空白图；建议使用 X11 或配置可用的 portal
- 远程桌面/录屏软件：某些安全策略可能导致 `desktopCapturer` 返回空缩略图，需要结合日志定位原因
